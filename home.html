  • x64 (64-bit) Processor 1-2 GHz (Required)
  • Windows 10 (build 1903+) or Windows 11
  • Intel i3 or AMD equivalent
  • 4GB RAM minimum
  • 500MB free disk space

First Run

  • Run as Administrator (recommended)

Configuration Files

  • Accounts.txt - User account information

Repair & Maintenance

Comprehensive troubleshooting and maintenance solutions for optimal performance

Automatic Repair (Fetcher)

Use the Fetcher - Repair Setup feature for automatic error correction and file restoration with comprehensive diagnostics. This tool can automatically detect and fix common issues.

Manual Repair (Shell Integrator)

Complete manual repair process with step-by-step commands for advanced troubleshooting and system restoration. Includes detailed command sequence for expert users.

Display Issues

Adjust DPI settings for high-resolution displays, set zoom to 100%, and update graphics drivers for optimal performance. Configure high DPI settings for laptops.

Performance Issues

Run as Administrator, close unnecessary applications, and ensure minimum system requirements are met. Monitor system resources for optimal performance.

Network Connectivity

Check internet connection for file operations and network tools, verify firewall settings, and ensure proper network permissions.

File Access Issues

Verify administrator privileges, check file permissions and ownership, ensure adequate disk space for operations and proper security settings.
